Student Grades Analysis with NumPy
Overview
This project utilizes NumPy to simulate student grades in three subjects (Math, English, Science) and perform basic analysis. It calculates average grades for each student, identifies the student with the highest average grade, and analyzes the performance in each subject.

Installation
Ensure you have Python installed. You can download it from python.org.

Install NumPy using the following command:

bash
pip install numpy
Download or clone this repository.

Usage
Open a terminal and navigate to the project directory.

Run the script:

bash
python student_grades_analysis.py
View the results displayed in the terminal.
